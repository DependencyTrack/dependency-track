**Features:**

* Improve performance of vuln data source mirroring - [apiserver/#6040]
* Handle epoch PURL qualifier for version comparison - [apiserver/#6083]
* Add project filters to component identity search - [apiserver/#6087]

**Fixes:**

* Fix incorrect format of Composer packages being sent to Trivy - [apiserver/#6117]
* Fix incorrect version comparison of NuGet packages when v-prefix is present - [apiserver/#6116]

For a complete list of changes, refer to the respective GitHub milestones:

* [API server milestone 4.14.2](https://github.com/DependencyTrack/dependency-track/milestone/64?closed=1)
* [Frontend milestone 4.14.2](https://github.com/DependencyTrack/frontend/milestone/36?closed=1)

We thank all organizations and individuals who contributed to this release, from logging issues to taking part in discussions on GitHub & Slack to testing of fixes.
