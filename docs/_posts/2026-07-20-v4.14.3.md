**Features:**

* Improve VEX import performance - [apiserver/#6141]
* Include `groupBy` parameter in DefectDojo finding upload - [apiserver/#6130]
* Include analysis detail in DefectDojo finding payload - [apiserver/#6181]
* Improve performance of OSV mirroring - [apiserver/#6345]
* Show exploit predictions from all sources - [frontend/#1523]

**Fixes:**

* Reject parent objects with null UUID when creating, updating or patching projects - [apiserver/#6355]
* Fix NPE and redundant queries in VEX / VDR export - [apiserver/#6614]
* Prevent duplicate downloads when using export and download buttons - [frontend/#1659]

For a complete list of changes, refer to the respective GitHub milestones:

* [API server milestone 4.14.3](https://github.com/DependencyTrack/dependency-track/milestone/65?closed=1)
* [Frontend milestone 4.14.3](https://github.com/DependencyTrack/frontend/milestone/48?closed=1)

We thank all organizations and individuals who contributed to this release, from logging issues to taking part in discussions on GitHub & Slack to testing of fixes.
